## Cómo reemplazar una imagen

1. Subí el original (PNG/JPG) y convertilo a WebP con el mismo nombre (`caso-N.webp`).
   Recorte recomendado: cuadrado tomado desde arriba (el hero), ~900×900, calidad 80.
2. Mantené los nombres `caso-1..4.webp` para no tocar el HTML.

Si una imagen falta, la tarjeta muestra un placeholder elegante en vez de romperse.
